### 检索问题示例

"补考和重修有什么区别" → Top 5 包含「补考和重修管理规定」和「缓考管理规定」。后者是 Chroma 向量误匹配，但因 BM25 权重高，最终排序影响小。

**建议**：切换为 bge-m3 或 text2vec-large-chinese 中文 embedding 模型。

## 四、已知限制

1. **向量中文差**：Chroma ONNX all-MiniLM-L6-v2 是英文模型，需替换
2. **关键词误报**："学位证和毕业证区别" 被误判为 high（含"学位"关键词）
3. **长 chunk 残留**：学生手册 25 段 >800 中文字符，为连续表格无自然断点
4. **无 LLM 时无法区分无依据和系统错误**：两者都返回 "系统暂时不可用"
5. **DeepSeek API 偶发超时**：Q8 测试时经历 60s 超时后重试成功

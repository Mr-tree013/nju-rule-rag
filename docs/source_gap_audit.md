## A. AI 使用与学术诚信 ⭐⭐⭐⭐⭐

### 核心官方文件

**《关于本科生规范使用生成式人工智能工具的指导意见（试行）》**
- 文号：南本院〔2025〕22 号
- 发布：2025年3月27日
- 部门：南京大学本科生院
- 来源：https://jw.nju.edu.cn
- 相关报道：https://news.nju.edu.cn

### 一、适用对象与应用场景

**适用对象**：全校本科生（含交流生、留学生）。文件同时对指导教师和任课教师在毕业论文和教学中的监管职责有延伸规定。

**应用场景**覆盖全过程：
- 日常课程作业、随堂测试、期末论文/报告
- 学科竞赛、创新创业训练计划（国创/省创）
- 本科毕业论文（设计）的开题、研究、撰写、答辩全过程
- 推免、评优等涉及学术成果提交的场景

---

### 二、禁止代写的核心内容（红线）

| 类别 | 禁止内容 |
|------|---------|
| **论文全篇** | 严禁直接使用 AI 生成毕业论文（设计）的摘要、正文、核心结论或代码 |
| **数据与图表** | 严禁利用 AI 凭空伪造、篡改或润色实验数据、调研结果、统计图表及文献引用 |
| **课程作业** | 严禁直接复制 AI 生成的文本、代码作为日常作业、编程练习或课程小论文提交 |
| **引用列表** | **严禁直接生成学术引用与参考文献列表** |
| **研究假设** | 禁止代写研究（实验）假设提出、选题意义 |
| **方案设计** | 禁止代写方案设计、创新性方法设计 |
| **研究结果** | 禁止代写研究（实验）数据、研究（实验）结果 |
| **分析结论** | 禁止代写分析与讨论、结论总结 |

---

### 三、允许使用的合规范围 + 申请审批流程

**合规使用范围（须教师审批）：**

| 允许范围 | 说明 |
|---------|------|
| 文献与资料检索 | 寻找研究线索、翻译外文文献、梳理文献大意 |
| 语言润色与纠错 | 对学生自己撰写的文本进行语法纠错、语句通顺度优化（不得改变原意） |
| 代码调试与辅助 | 排查程序 Bug、理解算法逻辑（而非直接生成完整项目） |
| 激发灵感 | 选题阶段的头脑风暴、拓宽研究视角 |

**申请审批流程：**
- **毕业论文阶段**：开题前向指导教师汇报计划使用何种 AI 工具，并在开题报告中明确写明
- **日常课程**：遵循"任课教师负责制"，教师有权决定是否允许。如允许，学生需在提交作业前向教师口头或书面报备
- **核心原则**：须征得任课教师**书面/电子签名同意**

---

### 四、使用留痕与透明披露要求

原则：**"谁使用、谁披露、谁负责"**

**全过程留痕：**
- 必须保留原始提示词（Prompts）和 AI 生成的原始输出记录
- 形式可以是：截图、对话链接或文本存档
- 供学校随时抽查和学术不端鉴定使用

**透明披露要求（毕业论文等)：**
文末或指定位置必须明确披露——
1. 使用的生成式 AI 工具名称及版本（如 ChatGPT-4o、文心一言 4.0）
2. 使用的具体时间段
3. AI 在成果中发挥的具体作用（如"用于第三章英文文献翻译""用于附录代码 Debug"）

---

### 五、违规处理的分级处分体系

| 行为类型 | 处分 |
|---------|------|
| 日常作业/竞赛违规 | 相关作业计 0 分、取消竞赛资格、学院通报批评或约谈 |
| 毕业论文答辩前发现 | 责令推迟答辩、论文降级、情节严重者取消答辩资格延期毕业 |
| 毕业论文答辩后发现 | 经学术委员会认定学术不端 → 撤销已授予的学士学位、注销学历证书 |
| 纪律处分等级 | 警告 → 严重警告 → 记过 → 留校察看 → **开除学籍**，记入个人档案 |

### 六、AI 使用声明模板

文件附录提供统一模板：**《南京大学本科生生成式人工智能（GAI）工具使用声明表》**

模板包含以下要素（学生签字确认）：

| 要素 | 内容 |
|------|------|
| 学生基本信息 | 姓名、学号、院系、专业 |
| 成果信息 | 课程名称/论文题目、指导教师/任课教师姓名 |
| AI 工具信息 | 工具名称、版本号、开发公司 |
| 使用详情清单 | 表格形式：列出应用在哪些章节/段落、具体用途（数据处理、翻译、润色）、**提示词示例** |
| 诚信承诺 | 学生签字，承诺除声明部分外均为独立完成 |
| 导师/教师意见 | 指导教师或任课教师签字确认 |

### 覆盖的问题确认

> ✅ 课程论文能不能用 ChatGPT？→ "任课教师负责制"，须教师审批
> ✅ AI 润色英文是否算学术不端？→ 语言润色属于合规范围，但须披露
> ✅ 毕设代码用 AI 是否需要声明？→ 是，必须填使用声明表
> ✅ 学校有无统一模板？→ 有，文件附录提供
> ✅ 只用 AI 改语法算违规吗？→ 须教师审批准许，且须披露
> ✅ 小组作业一人用 AI 其他人受影响？→ 需查具体规定；原则上是使用者个人负责

### 一、本科生实验准入：安全培训与考核

**原则**："不培训不准入，不考核不准入"

**安全培训内容：**

| 类别 | 具体内容 |
|------|---------|
| 通用安全 | 消防安全（灭火器使用、逃生路线）、用电安全、防盗及内保、急救常识 |
| 专业安全 | 按学科：危化品使用、生物安全、辐射安全、特种设备（高压灭菌锅、气瓶） |
| 环境制度 | 目标实验室规章制度、危险源分布、PPE 正确穿戴、废弃物分类处置 |

**安全考核内容：**
- 平台："南京大学实验室安全教育与考试系统"（线上）
- 形式：单选 + 多选 + 判断题
- **合格线：90 分**（总分 100）
- 线下确认：考试合格后，接受实验室负责人现场安全交底，签署《实验室安全承诺书》

---

### 二、危化品"五双"管理制度

| 原则 | 含义 |
|------|------|
| **双人保管** | 保险柜/库房配两把不同锁，两名不同保管员分别管理钥匙/密码 |
| **双人领取** | 每次领用需两人在场签字、互相监督 |
| **双人使用** | 实验过程中必须两名实验人员共同在场 |
| **双把锁** | 专柜或仓库安装两把锁，两人同时到场才能开 |
| **双本账** | 两套独立台账（电子+纸质），账账相符、账物相符、流向可追溯 |

---

### 三、实验室事故上报流程

```
事故发生 → 立即自救/互救
    │
    ▼ (即时)
1. 现场口头报告导师/实验室负责人
    │
    ▼ (1小时内)  
2. 学院分管领导及安全员上报
    │
    ▼ (2小时内)
3. 国有资产管理处（实验室安全管理科）
   + 安全保卫处（消防/治安）
   + 校医院（伤员救治）
    │
    ▼
4. 学校成立事故调查组（国资处牵头 + 安保处 + 纪检监察 + 学院）
   → 查明原因 → 明确责任 → 整改措施
```

### 四、设备损坏赔偿标准

| 情形 | 处理 |
|------|------|
| **非责任事故** | 设备质量问题、自然老化、不可抗力、按章操作下的合理损耗 → 不赔 |
| **责任事故** | 违反操作规程、擅自使用、粗心大意、玩忽职守、私自拆卸 → 须赔 |
| 轻微疏忽且主动报告 | **30-50%** 维修费 |
| 严重违章/瞒报 | 较高比例甚至**全额赔偿**，无法修复按折旧重置价赔偿 |
| 重大财产损失 | 经济赔偿 + 纪律处分 + 可能法律责任 |

---

### 五、夜间独自实验规定

- **绝对禁止**：高危实验（易燃易爆品、剧毒品、高温高压、射线装置）**绝对禁止**单人夜间（22:00-6:00）或节假日操作
- **一般实验**需满足：
  1. 导师书面/线上审批
  2. **双人值守**（至少 2 人同区域内）
  3. 楼宇管理员处登记

---

### 六、校外实习实验事故责任

| 实习类型 | 责任方 |
|---------|--------|
| 学校统一安排的集中实习/联合培养 | 学校协助处理，主要责任由提供场地的**校外接收单位**承担 |
| 学生自主联系的校外实习（未签三方协议） | 责任由**学生个人**及**接收实习单位**承担，学校不承担法律直接赔偿责任 |
| 保险保障 | 学校统一购买学平险/实习责任险，可通过商业保险理赔 |

### 一、三方协议解约流程

1. **协商**：联系原单位获得《解约函》（加盖公章）
2. **91job 平台**：登录 → "办事大厅" → "网签/解约申请" → 填写原因 + 上传解约函
3. **审核链**：辅导员/院系初审 → 就业指导中心终审 → 系统恢复"未签约"状态
4. **违约金**：依约支付给原单位（升学/入伍/国家基层项目可申请免除）

| **隔离保护** | 导师更换机制（研究生）、宿舍调换+班级隔离（学生间欺凌） |
| **心理援助** | 心理健康教育与咨询中心免费长期专业疏导 |

### 一、走读/校外住宿申请流程

> **⚠️ 核心原则**：未经批准擅自校外住宿属于**违纪**，面临纪律处分。

| 步骤 | 操作 |
|------|------|
| 1. 提交书面申请 | 向院系提交走读/校外住宿申请 + 证明材料 |
| 2. 证明材料 | 南京本地户口簿 / 医院疾病诊断书 / 家长知情同意书 |
| 3. 院系审核 | 辅导员 + 学工负责人 + 导师（如有）签字 + 院系公章 |
| 4. 资产处备案 | 携带审批表至公寓物业管理中心或资产管理处住房管理科办退宿 |
| 5. 费用清算 | 财务部门按实际住宿时间清算或停收住宿费 |
| 6. 安全闭环 | 校园卡门禁权限调整，学生自负校外人身财产安全 |

---

### 二、延毕住宿申请

| 要点 | 说明 |
|------|------|
| 原则 | 正常学制到期（6月底）后统一退宿，**不自动顺延** |
| 申请入口 | 南京大学网上办事服务大厅（ehall.nju.edu.cn） |
| 申请时间 | 6月下旬至7月上旬，关注资产处通知 |
| 审批后 | 另行集中安排/调整宿舍，规定时间内搬迁至新寝室 |
| 费用 | 按学期继续缴纳住宿费 |

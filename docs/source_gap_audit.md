**《关于本科生规范使用生成式人工智能工具的指导意见（试行）》**
- 文号：南本院〔2025〕22号
- 发布：2025年3月27日，南京大学本科生院
- 来源：https://jw.nju.edu.cn

### AI 回答摘要（五大要点）

| 要点 | 内容 |
|------|------|
| **适用对象** | 全校所有本科生，覆盖毕业论文、课程作业、实习实验报告、课外作品等所有考核环节 |
| **核心红线** | 禁止 AI 代写：研究假设提出、方案设计、研究数据/结果、分析讨论、结论总结、**直接生成学术引用与参考文献列表** |

## E. 性骚扰/欺凌/歧视 ✅ 可补充

### 核心文件

| 文件 | 适用对象 | 关键内容 |
|------|---------|---------|
| **《南京大学学生违纪处分规定》** | 加害者为学生 | 侮辱、恐吓、骚扰、欺凌，记过至开除学籍 |
| **《南京大学学生申诉处理办法》（2024）** | 受害学生 | 法定申诉权利、复查机制、处理时限 |
| **《南京大学师德失范行为处理暂行办法》** | 加害者为教师 | 性骚扰、不正当关系属于严重师德违规 |
| **《南京大学学生手册》** | 全体学生 | 违纪行为界定与处分程序 |

### 投诉流程
1. **举报渠道**：纪检监督电话 025-83592586，或向院系党委、保卫处、学工处、法学院的师德师风监督办公室递交书面材料
2. **匿名举报**：原则上要求实名；但匿名举报如提供具体事实和确凿证据，同样受理
3. **处理流程**：受理登记 → 启动调查（独立客观） → 结果审核（校长办公会） → 处分下达 → 申诉期
4. **保护措施**：隐私保密、防止打击报复、教学与生活隔离（导师更换/宿舍调换）、心理援助支持

### 建议操作
1. 获取上述 4 份文件的全文，放入知识库
2. 这是**高风险类别**，回答只引用已有规定，不下结论
3. 建议新增 `source_type=high_risk_sensitive`，检索时放在高风险通道

## F. 校外租房/走读/住宿例外 ✅ 可补充

### 核心文件

| 文件 | 出处 | 关键内容 |
|------|------|---------|
| **《南京大学学生住宿、退宿须知》** | 资产管理处（2014年） | 退宿流程、校外住宿申请 |
| **《南京大学学生手册》** | 本科生院 | **未经批准擅自校外住宿属于违纪**，扣分+处分 |
| **2023年暑期学生宿舍安排及搬迁调整的通知** | 资产管理处 | 延毕住宿申请、公派出国住宿顺延 |
| **住房管理科制度** | 资产管理处 | 宿舍分配、退宿、调换的正式制度 |

### 走读/校外住宿流程（AI 回答摘要）
1. 提交书面申请 + 证明材料（户口簿/医院证明/家长知情同意书）
2. 院系审核签字（辅导员 + 学工负责人 + 导师）
3. 资产管理处住房管理科备案 + 退宿 → 住宿费清算
4. 校外安全闭环（门禁权限调整，学生自负人身财产安全）

### 延毕住宿
1. 毕业生统一退宿清理（6月底）
2. 延毕生须在暑期关注资产处通知，登录 ehall.nju.edu.cn 递交延期住宿申请
3. 学校视房源另行集中安排，需在规定时间内搬迁
4. 继续缴纳住宿费

### 建议操作
1. 获取《南京大学学生住宿、退宿须知》全文
2. 补充资产管理处住房管理科最新联系方式
3. 注意：2014年文件可能需要确认是否最新版本

# CSC国家留学基金委项目与语言考试指南

> 来源：南京大学学生交流办公室 (stuex.nju.edu.cn)、南京大学人力资源处 (hr.nju.edu.cn)、国家留学基金委 (csc.edu.cn)

## CSC项目概述

国家留学基金委（CSC）每年资助高校学生和教师赴海外学习/研究。南京大学作为重点高校，可推荐学生申请多个CSC项目。

## 常见CSC项目类型

| 项目类型 | 面向对象 | 说明 |
|---------|---------|------|
| 国家公派研究生项目 | 硕士/博士生 | 赴国外攻读学位或联合培养 |
| 本科生实习奖学金 | 本科生 | 如加拿大Mitacs、阿尔伯塔大学实习等 |
| 互换奖学金项目 | 本科生/研究生 | 与特定国家互换（如中加学者交换项目、芬兰互换奖学金等） |
| 非通用语种人才支持计划 | 相关专业学生 | 特定语种人才培养 |
| 青年骨干教师出国研修项目 | 教师 | 青年教师出国访学 |

## CSC项目通用要求

### 基本条件
- 中国公民，无国外永久居留权
- 南京大学正式在册学生/教职工
- 满足各项目的年龄、学历等基本要求
- 身心健康

### 外语水平要求

申请CSC项目必须满足国家留学基金委的**外语水平要求**，常见达标方式：

| 考试/条件 | 要求 | 有效期 |
|-----------|------|--------|
| **雅思（IELTS）学术类** | 6.5分及以上 | 2年 |
| **托福（TOEFL iBT）** | 95分及以上 | 2年 |
| **全国外语水平考试（WSK/PETS5）** | 笔试总分55分以上，听力18分以上，口试3分以上 | 2年 |
| **教育部指定出国培训部** | 英语高级班结业证书 | 视项目而定 |
| **国外留学/工作经历** | 同一语种国家留学一学年（8-12个月）或连续工作一年以上 | 需提供证明 |

### 重要说明
- 部分项目**允许申请时外语未达标**，但必须提供已有考试成绩作为证明，并在**派出前**达到合格要求
- 具体外语要求以CSC官方规定为准：https://www.csc.edu.cn/article/2514

## 申请流程

1. 关注南京大学国际处/学生交流办公室发布的遴选通知
2. 通过**国家公派留学管理信息平台** (http://sa.csc.edu.cn) 网上报名
3. 提交申请材料至学校审核
4. 学校推荐 → CSC评审 → 录取通知

## 校内申请渠道

- **本科生项目**：南京大学学生交流办公室 (https://stuex.nju.edu.cn)
- **研究生项目**：南京大学研究生院
- **教师项目**：南京大学人力资源处 (https://hr.nju.edu.cn)

## 语言考试准备资源

### 校内资源
- 南京大学外国语学院提供语言培训信息
- 部分院系有出国交流QQ群/微信群，可获取学长学姐经验

### 考试类型速查

| 考试 | 适用方向 | 说明 |
|------|---------|------|
| 雅思（IELTS） | 英联邦/欧洲 | 最广泛接受 |
| 托福（TOEFL） | 北美为主 | 全球通用 |
| GRE | 美国研究生院 | 理工科/社科博士申请 |
| GMAT | 商学院 | MBA/商科硕士 |
| 日语 JLPT | 日本 | N1/N2级别 |
| 德语 TestDaF | 德国 | 4×4 |
| 法语 DELF/DALF | 法国 | B2/C1级别 |

## 护照与签证办理

### 护照办理
- 首次办理：携带身份证至户籍所在地公安局出入境管理大厅
- 在校生（户口已迁至学校）：可在南京办理，需提供在校证明
- 办理周期：一般7-15个工作日

### 签证办理
- 根据目的地国家要求准备材料
- 部分国家要求面签（如美国）或通过签证中心（如申根国家）
- 建议提前2-3个月开始准备

## 关键提示

1. **外语成绩有效期为2年**，申请时必须在有效期内
2. 建议**大二下学期或大三上学期**考出语言成绩
3. CSC项目的具体条件以每年发布的遴选通知为准
4. 关注南京大学国际处、学生交流办公室的最新通知

### 一、本科生实验准入：安全培训与考核

**原则**："不培训不准入，不考核不准入"

**安全培训内容：**

| 类别 | 具体内容 |
|------|---------|
| 通用安全 | 消防安全（灭火器使用、逃生路线）、用电安全、防盗及内保、急救常识 |
| 专业安全 | 按学科：危化品使用、生物安全、辐射安全、特种设备（高压灭菌锅、气瓶） |
| 环境制度 | 目标实验室规章制度、危险源分布、PPE 正确穿戴、废弃物分类处置 |

**安全考核内容：**
- 平台："南京大学实验室安全教育与考试系统"（线上）
- 形式：单选 + 多选 + 判断题
- **合格线：90 分**（总分 100）
- 线下确认：考试合格后，接受实验室负责人现场安全交底，签署《实验室安全承诺书》

### 二、危化品"五双"管理制度

| 原则 | 含义 |
|------|------|
| **双人保管** | 保险柜/库房配两把不同锁，两名不同保管员分别管理钥匙/密码 |
| **双人领取** | 每次领用需两人在场签字、互相监督 |
| **双人使用** | 实验过程中必须两名实验人员共同在场 |
| **双把锁** | 专柜或仓库安装两把锁，两人同时到场才能开 |
| **双本账** | 两套独立台账（电子+纸质），账账相符、账物相符、流向可追溯 |

### 三、实验室事故上报流程

```
事故发生 → 立即自救/互救
    │
    ▼ (即时)
1. 现场口头报告导师/实验室负责人
    │
    ▼ (1小时内)  
2. 学院分管领导及安全员上报
    │
    ▼ (2小时内)
3. 国有资产管理处（实验室安全管理科）
   + 安全保卫处（消防/治安）
   + 校医院（伤员救治）
    │
    ▼
4. 学校成立事故调查组（国资处牵头 + 安保处 + 纪检监察 + 学院）
   → 查明原因 → 明确责任 → 整改措施
```

### 四、设备损坏赔偿标准

| 情形 | 处理 |
|------|------|
| **非责任事故** | 设备质量问题、自然老化、不可抗力、按章操作下的合理损耗 → 不赔 |
| **责任事故** | 违反操作规程、擅自使用、粗心大意、玩忽职守、私自拆卸 → 须赔 |
| 轻微疏忽且主动报告 | **30-50%** 维修费 |
| 严重违章/瞒报 | 较高比例甚至**全额赔偿**，无法修复按折旧重置价赔偿 |
| 重大财产损失 | 经济赔偿 + 纪律处分 + 可能法律责任 |

### 五、夜间独自实验规定

- **绝对禁止**：高危实验（易燃易爆品、剧毒品、高温高压、射线装置）**绝对禁止**单人夜间（22:00-6:00）或节假日操作
- **一般实验**需满足：
  1. 导师书面/线上审批
  2. **双人值守**（至少 2 人同区域内）
  3. 楼宇管理员处登记

### 六、校外实习实验事故责任

| 实习类型 | 责任方 |
|---------|--------|
| 学校统一安排的集中实习/联合培养 | 学校协助处理，主要责任由提供场地的**校外接收单位**承担 |
| 学生自主联系的校外实习（未签三方协议） | 责任由**学生个人**及**接收实习单位**承担，学校不承担法律直接赔偿责任 |
| 保险保障 | 学校统一购买学平险/实习责任险，可通过商业保险理赔 |

### 一、走读/校外住宿申请流程

> **⚠️ 核心原则**：未经批准擅自校外住宿属于**违纪**，面临纪律处分。

| 步骤 | 操作 |
|------|------|
| 1. 提交书面申请 | 向院系提交走读/校外住宿申请 + 证明材料 |
| 2. 证明材料 | 南京本地户口簿 / 医院疾病诊断书 / 家长知情同意书 |
| 3. 院系审核 | 辅导员 + 学工负责人 + 导师（如有）签字 + 院系公章 |
| 4. 资产处备案 | 携带审批表至公寓物业管理中心或资产管理处住房管理科办退宿 |
| 5. 费用清算 | 财务部门按实际住宿时间清算或停收住宿费 |
| 6. 安全闭环 | 校园卡门禁权限调整，学生自负校外人身财产安全 |

---

### 二、延毕住宿申请

| 要点 | 说明 |
|------|------|
| 原则 | 正常学制到期（6月底）后统一退宿，**不自动顺延** |
| 申请入口 | 南京大学网上办事服务大厅（ehall.nju.edu.cn） |
| 申请时间 | 6月下旬至7月上旬，关注资产处通知 |
| 审批后 | 另行集中安排/调整宿舍，规定时间内搬迁至新寝室 |
| 费用 | 按学期继续缴纳住宿费 |

# 南京大学本科招生政策与录取规则

> 来源：南京大学2025年普通本科招生章程、南京大学本科招生网 (bkzs.nju.edu.cn)

## 学校概况

南京大学是教育部直属的全国重点综合性、研究型大学，国家"双一流"建设高校。拥有**鼓楼、仙林、苏州、浦口**四个校区。

## 录取规则

### 投档比例

| 投档方式 | 调档比例 |
|---------|---------|
| 平行志愿批次 | 原则上不超过 **105%** |
| 顺序志愿批次 | 原则上不超过 **120%** |

### 专业录取原则

- **分数优先、遵循志愿**：依据考生高考实考成绩（不含政策加分）进行专业录取
- **不设专业级差**：各专业志愿之间没有级差分
- **同分排序**：按各省提供的投档位次排序；未提供位次的省份，依次按 **数学 → 语文 → 外语** 成绩排序

### 调剂与退档

- 服从专业调剂 → 调剂到未录满专业
- 不服从专业调剂 → 作退档处理

### 政策加分

- 认可教育部规定的全国性政策加分，**最高不超过20分**
- 同一考生如有多项加分，取最高一项

## 招生计划

- **预留计划**：占年度招生计划总数的 **1%**，用于调节各地统考上线生源的不平衡
- **招生模式**：按专业和学科大类相结合的方式招生

## 特殊类型招生通道

| 招生类型 | 说明 |
|---------|------|
| 强基计划 | 8个专业，聚焦基础学科 |
| 江苏省综合评价 | 仅招物理类，高考成绩×85% + 南大测试×15% |
| 高校专项计划 | 面向农村学子，降分优惠 |
| 外语保送生 | 仅限南京外国语学校 |
| 艺术类 | 戏剧影视文学专业 |
| 中外合作办学 | 南赫学院（大气科学），学费7.5万元/学年 |

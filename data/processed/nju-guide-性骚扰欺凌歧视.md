### 二、处理流程与时间线

```
1. 受理登记（受理后即启动）
   核实被投诉对象信息 + 证据完整性
        │
        ▼
2. 调查阶段（独立客观调查）
   教职工类：师德师风建设领导小组
   学生类：学工处 + 团委 + 保卫处
        │
        ▼
3. 结果审核
   调查报告 → 校长办公会/党政联席会议审定
        │
        ▼
4. 处分下达 + 申诉期
   当事人不服 → 学生申诉处理委员会 → 法定期限内复查
```

### 三、受害者保护措施

| 措施 | 内容 |
|------|------|
| **隐私保密** | 学工、保卫、法务等须对举报人身份和隐私内容严格保密 |
| **反打击报复** | 严禁转嫁压力、威胁、变相报复，一经查实加重处分乃至移送司法 |
| **隔离保护** | 导师更换机制（研究生）、宿舍调换+班级隔离（学生间欺凌） |
| **心理援助** | 心理健康教育与咨询中心免费长期专业疏导 |

# 体测标准与体育课攻略

> 来源：南京大学本科生院网站 (jw.nju.edu.cn) — 公共体育课程学习须知

## 一、体育课基本要求

| 项目 | 要求 |
|------|------|
| 学分类型 | 必修课 |
| 总学分 | **4学分** |
| 修读年限 | 须在**前2年**内完成 |
| 选课限制 | 每学期限选**1门** |
| 跨校区 | 鼓楼/仙林**不可跨校区**选课 |
| 重修规则 | 前4次修读均属首次修读（不限项目），第5次起视为重修 |

> ⚠️ 体育课是申请免试研究生的应修课程，未选课需出具说明。

## 二、体育课成绩构成

| 项目 | 占比 |
|------|------|
| 专项课堂 | **40%** |
| 阳光体育锻炼 | **20%** |
| 基本身体素质 | **30%** |
| 平时成绩 | **10%** |

### 基本身体素质说明
- **第一学期**：国家体质健康测试（标准体测项目）
- **第二学期**：2400米 + 引体向上（男）/ 仰卧起坐（女）

## 三、体质健康测试（体测）

### 测试项目

| 性别 | 项目 |
|------|------|
| 男生 | 身高、体重、肺活量、坐位体前屈、50米、立定跳远、**引体向上**、**1000米** |
| 女生 | 身高、体重、肺活量、坐位体前屈、50米、立定跳远、**仰卧起坐**、**800米** |

### 评分等级

| 等级 | 分数范围 |
|------|----------|
| 优秀 | ≥ 90分 |
| 良好 | 80 ~ 89.9分 |
| 及格 | 60 ~ 79.9分 |
| 不及格 | ≤ 59.9分 |

### 毕业要求
- **毕业时体测总分 ≥ 50分**，否则按结业或肄业处理
- 毕业成绩 = 大四学年成绩 × 50% + 其他学年平均分 × 50%
- 每学年有一次补测机会
- 大四体测 ≥ 60分可获江苏省教育厅颁发的体质健康等级证书

### 测试时间
- **秋季学期（10–12月）**：统一体测
- **春季学期（3–5月）**：补测

## 四、特殊规定

### 保健课
- 患有心血管病、哮喘、癫痫等不适合剧烈运动的学生须选保健课
- 成绩统一按**60分**登记
- 需提供医院证明

### 免测
- 因病或残疾可在**南京大学APP**体测板块申请免测
- 需提交医院证明，经审核通过后生效

### 缺课与缓考
- 无故缺课达到**4次** → 课程不及格，不予补考，须重修
- 缓考须在考前向学院提交申请，经本科生院审批

# 境外交换项目指南

> 来源：南京大学本科生院网站 (jw.nju.edu.cn)、南京大学学生交流办公室 (stuex.nju.edu.cn)

## 交换项目概述

南京大学定期遴选本科生赴境外签约合作院校进行交换学习，旨在拓展学生视野、促进专业发展、提升跨文化交流能力。

## 交换生类型

| 类型 | 说明 |
|------|------|
| **交流生（Exchange Students）** | 免学费，名额有限 |
| **访问学生（Visiting / Fee-paying Students）** | 自费 |

共同点：交换生只能在本校获取学位。

## 项目时长分类

| 类型 | 时长 |
|------|------|
| 短期项目 | 少于3个月（寒暑假项目等） |
| 长期项目 | 3个月以上，通常为一学期或一学年 |

## 申请条件

1. **身份条件**：南京大学普通全日制本科学生
2. **时长限制**：在校期间境外交流学习时间**累计不得超过一年**
3. **项目限制**：三个月以上、免学费、有名额的长期项目，**原则上只能参加一次**
4. **诚信约束**：获得推荐和录取后不得擅自放弃；无故放弃者将取消本科期间一切校级交换资格

## 申请流程（共5步）

### 第一步：校内报名
1. 通过国际处项目通知页面（https://stuex.nju.edu.cn）查看项目信息
2. 登录**南京大学交换生管理系统**（jw.nju.edu.cn → 学生平台 → 交换生管理系统）
3. 提交网上申请
4. 经院系辅导员、教学院长线上审核
5. 在线打印承诺书、手动抄写并签名后上传
6. 本科生院负责遴选，入围名单在系统首页公示

### 第二步：对方网申
学校向外方提名候选人后，根据邮件通知准备材料，完成外方网申或材料递交。部分外方学校会组织面试或二次选拔。

### 第三步：OA审批
所有项目均需在**项目开始前一个月**于学校OA系统完成出国（境）审批流程。

### 第四步：校内学业事项处理
- **交换开始前**：开学两周内在教服平台完成交流备案
- **交换期间**：一般不得办理本校选课注册手续；每学期最多可申请两门"免修不免考"课程
- **课程退选**：应主动于开课两周内退课或申请缓修

### 第五步：返校后学分转换
1. 打印《交流课程学分认定申请表》
2. 附成绩单原件及课程介绍、学时证明
3. 院系审核签字盖章
4. 本科生院复审并录入成绩

## 时间安排关键节点

| 事项 | 截止时间 |
|------|----------|
| OA出国审批 | 项目开始前一个月 |
| 教服平台备案（学期交换） | 我校开学两周内 |
| 教服平台备案（寒暑假项目） | 项目开始前 |
| 退课或申请缓修 | 开课两周内 |
| 学分转换申请（长期项目） | 返校后一年内 |

## 成绩认定规则

- 与院系培养计划对应的课程 → 按内容认定性质并转换学分
- 无法对应的课程 → 统一认定为选修课
- 符合通识教育理念和标准的 → 经校通识教育委员会审核可认定为通识课程

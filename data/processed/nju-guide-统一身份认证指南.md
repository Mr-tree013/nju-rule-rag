# 南京大学统一身份认证指南

> 来源：南京大学信息化建设管理服务中心 (itsc.nju.edu.cn)

## 什么是统一身份认证

统一身份认证是南京大学的**单点登录（SSO）系统**，让你使用**同一套账号和密码**登录所有校内系统。

### 覆盖的系统

| 系统类别 | 具体名称 |
|----------|----------|
| 网络认证 | p.nju.edu.cn（上网登录） |
| VPN | Easy Connect 客户端（校外访问校内资源） |
| 微信 | 公众号「南京大学信息门户」 |
| APP | 「南京大学」APP |
| 网上办事大厅 | ehall.nju.edu.cn |
| 教务系统 | 本科生/研究生教务系统 |
| 其他 | OA办公、迎新系统、就业系统、学生第二课堂等 |

### 如何识别统一身份认证页面
- 登录界面网址以 `http://authserver.nju.edu.cn/` 开头
- 页面提示"使用统一身份认证登录"

## 账号与密码

### 用户名
**学号**（本科生/研究生）/ 工号（教职工）/ 自管号（临时人员）

### 账号激活（重要！2026年新变化）

2026年2月7日后新开通的账号，**首次使用前必须手动完成"账号激活"**。

#### 移动端激活
访问：https://authserver.nju.edu.cn/retrieve-password/activationMobile/index.html#/home

#### 电脑端激活
访问：https://authserver.nju.edu.cn/retrieve-password/accountActivation/index.html#/

> 注意：姓名及证件类型必须与学校源头系统（人事/研究生/教务学工/海外教育学院）录入的信息一致，否则可能激活失败。

### 初始密码（2026年2月7日前的老账号）
- 全部身份证件号（港澳台侨学生为通行证号）
- 或 身份证号后六位
- 如有短信/页面通知初始密码，以通知为准

## 绑定手机号（必须操作！）

自2022年9月起，统一身份认证登录需要**手机短信验证**（半年内只需一次）。必须提前绑定手机号。

### 绑定方法一：微信公众号
1. 微信关注 **「南京大学信息门户」**
2. 点击「个人中心」
3. 设置统一身份认证手机号

### 绑定方法二：南京大学APP
1. 安装「南京大学」APP
2. 点击「学工号」登录
3. 在「服务」页面搜索 **「认证账号设置」**
4. 绑定手机号
> ⚠️ 注意：APP本身的"账号设置"设置的是APP信息，**不是**统一身份认证信息！要搜索"认证账号设置"

### 绑定方法三：扫码登录后设置（电脑端）
1. 打开统一身份认证平台：http://authserver.nju.edu.cn/authserver/login
2. 手机微信/APP扫码登录
3. 登录后设置手机号，并**尽快修改初始密码**（建议字母+数字，8位以上）

功能：
- 账号激活
- 找回密码
- 修改密码

- 账号绑定管理

### 密码不正确怎么办？
参见详细FAQ：https://itsc.nju.edu.cn/ee/60/c21475a585312/page.htm#faq2

**原因1**：手机号未绑定或绑定错误 → 登录自助服务中心确认
**原因2**：退订了来自 106904000520 的短信通知 → 需通过OA申请恢复订阅，咨询电话 89683791

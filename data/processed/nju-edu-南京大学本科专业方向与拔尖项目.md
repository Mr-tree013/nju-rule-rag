## 三、拔尖计划——匡亚明学院"最强理科班"

### 定位

教育部拔尖计划1.0与2.0基地，南大拔尖创新人才培养核心基地。

### 专业方向

理科大类（数学、物理、化学、生物、计算机及交叉学科），**低年级不分专业**。

### 培养特色

| 特色 | 说明 |
|------|------|
| **100%自由分流** | 学生可自由选择分流方向 |
| **100%自由选导师** | 自由选择学科导师和学业导师 |
| **100%自主申请国际交流** | 自主申请各类国际交流项目 |
| **100%入选拔尖计划** | 所有学生均入选"拔尖计划"培养 |
| **深造率80%+** | 毕业生进入本校及国内外名校深造 |

### 2025年改革

- 持续扩大招生规模，**在江苏投放计划接近去年3倍**
- "让基础更基础、让前沿更开放、让实践更真实、让学习更自由"
- 入校后可参加二次选拔

## 四、强基计划（2025年）

### 招生专业

| 科类 | 招生专业 | 选科要求 |
|------|----------|----------|
| 理科 | 数学与应用数学、信息与计算科学、物理学、化学、生物科学 | 物理+化学 |
| 文科 | 汉语言文学（古文字学）、历史学、哲学 | 不限 |

### 2025年新增两个培养方向

#### 1. 数学与应用数学 → 「智能科学」方向

- 依托**人工智能学院**培养
- 由欧洲科学院院士**周志华教授**领衔
- 计算机软件新技术全国重点实验室（连续3次获评计算机类第一）
- **本科生大三前**即可全部进入课题组
- 可进入**华为**等头部企业实训

#### 2. 物理学 → 「电子科学」方向

- 依托**集成电路学院**培养
- 汇聚**2位院士**在内的雄厚师资
- 配备**6000㎡以上**集成电路实训平台
- 联袂顶尖企业开设**20余门**实训课程
- 支持前往国际一流名校访学

### 培养四大支柱

1. **动态进出机制**——多阶段科学化考核
2. **通专融合课程体系**——通识教育 + 跨学科研修 + 本研衔接培养
3. **科研训练体系**——前沿引领、大师引路、国内外交流
4. **支持服务体系**——爱国教育、个性化指导、批判性思维培养

### 本研衔接

研究生阶段可在基础学科深耕，也可在**高端芯片与软件、智能科技、新材料、先进制造、国家安全**等关键领域进行学科交叉培养。

## 五、"至诚计划"——新工科拔尖培养（2025年新推出）

### 涵盖专业

人工智能、计算机科学与技术、集成电路设计与集成系统、软件工程

### 培养模式

- **"3+1+X"本研一体化**：本科科教融合 + 研究生"校企双导"产教融合
- **院士领衔**，知名学者担任导师
- 国家级科研平台 + 头部企业实训平台 + 国际一流训练项目
- 培育**战略型科技领军人才**

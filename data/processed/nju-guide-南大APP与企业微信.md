# 南京大学APP与企业微信指南

> 来源：南京大学信息化建设管理服务中心 (itsc.nju.edu.cn)

## 一、南大APP

### 下载方式

| 系统 | 下载方式 |
|------|---------|
| **iOS** | App Store 搜索"南京大学" |
| **Android** | 扫描信息化中心官网二维码下载 |
| **鸿蒙 5.0+** | 华为应用商店搜索"南京大学" |

### 登录方式
- 首次使用：点击右下角「学工号」→ 通过统一身份认证登录 → 完成手机号绑定
- 再次登录：可使用手机号或统一身份认证直接登录
- 验证短信以"【南京大学】"作为前缀发送

### 主要功能
- 入校填报、健康打卡
- 课表查询、成绩查询
- 校园卡服务（充值、查询、挂失）
- 网络服务（网费充值）
- 认证账号设置
- 体测申请（免测等）
- 校园生活各类服务

### 账号管理
- **更换学工号**：APP内"我的" → "设置" → "退出当前账号"可解绑原学号
- **手机号冲突**：用原手机号登录 → "我的" → "设置" → "校内身份管理" → 选择当前身份 → 右上角"编辑" → "删除"
- **忘记密码**：用户名是学工号，通过自助服务中心找回

### 报修与支持
- 电话：025-89683791
- 微信端：「南京大学信息门户」→ "我" → "我要反馈"提交工单

## 二、南京大学信息门户（微信公众号/企业微信）

### 关注方式
微信搜索并关注公众号 **「南京大学信息门户」**，扫码即可。

### 核心功能
- **移动教务助手**：查询成绩、课表、借教室
- **在线充值与缴费**：缴电费、网费、校园卡充值
- **消息推送**：上课提醒、成绩提醒、图书到期、卡余额不足、网费不足、电量不足
- **校园卡服务**：余额查询、挂失
- **认证账号设置**：绑定统一身份认证手机号

### 绑定方式
关注公众号后，使用统一身份认证登录即可完成绑定。

## 四、选课系统

### 入口
- **教服平台**：通过本科生院网站 (jw.nju.edu.cn) → 学生平台 → 教服平台
- 校外访问需先连接VPN

### 教服平台主要功能
| 功能 | 说明 |
|------|------|
| 学期选课 | 每学期选课/退课/补选 |
| 课表 | 查看学期课程安排 |
| 教学评估 | 期末课程评价 |
| 考务查看 | 查看考试安排 |
| 成绩查看 | 查看各学期成绩 |

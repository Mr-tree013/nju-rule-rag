# 校园网络与正版软件指南

> 来源：南京大学信息化建设管理服务中心 (itsc.nju.edu.cn)

## 一、校园网络接入

### 接入方式

#### 有线网络
- 设备设为DHCP自动获取IP
- 连接校园网有线接口即可访问校内资源
- 详细配置见FAQ

#### 无线网络（NJU-WLAN）
- 搜索并连接SSID为 **"NJU-WLAN"** 的信号
- 连接后即可访问校内网络资源
- 覆盖教学楼、图书馆、宿舍、食堂等全校区域

### 上网认证

接入网络后，打开浏览器访问：**http://p.nju.edu.cn**

使用统一身份认证登录，通过后即可访问校外互联网。

### 无感知认证

开启该功能后，设备进入NJU-WLAN覆盖区或连接有线网时，**无需每次认证登录，直接自动联网**。

设置路径：登录 p.nju.edu.cn → 开启无感知认证

### 多终端并发

支持最多**4台设备同时在线**（自2021年3月起），资费与原单终端相同。

### 上网资费

| 项目 | 说明 |
|------|------|
| 免费时长 | 每月赠送免费时长 |
| 封顶 | **20元/月**封顶 |
| 充值方式 | 微信信息门户或南大APP → 网络服务 → 网费充值 |

### Eduroam（全球学术WiFi漫游）

- 南大师生到**其他国内外高校**可搜索"eduroam"信号无线上网
- 使用完整的南大邮箱地址（学号@smail.nju.edu.cn）和统一身份认证密码登录
- 其他高校师生来访也可使用南大的eduroam

### 网络报障

- 电话：025-89683791
- 在线报障：微信信息门户 → 我要反馈

# 南京大学本科培养体系

> 最后更新：2026-05-29 | 信息来源：南大官网、本科生院(jw.nju.edu.cn)、南大招生小蓝鲸公众号

## 一、"三三制"人才培养模式（2009至今）

南京大学2009年率先提出的本科人才培养模式，获**国家级教学成果特等奖**。

### 第一个"三"——三个阶段

| 阶段 | 内容 |
|------|------|
| **大类培养阶段** | 新生按学科大类进入"书院"学习，修读通识教育课程、通修课程及学科基础课程 |
| **专业培养阶段** | 进入具体专业，夯实学科专业基础 |
| **多元培养阶段** | 根据学生发展意向分流培养 |

### 第二个"三"——三条路径

| 路径 | 方向 |
|------|------|
| **专业学术类** | 培养学术研究人才，提升知识探究高度 |
| **交叉复合类** | 打破学科壁垒，跨学科融合 |
| **就业创业类** | 培养职场精英和创新创业人才 |

### 核心特色——"三个百分百可选择"

- **课程百分百可选择**：学生自主定制课表
- **专业百分百可选择**：入学后通过学科准入、专业准入机制自由选择专业
- **发展路径百分百可选择**：三个方向自由切换

每位学生可以拥有一个**与众不同的个性化课表**。

## 二、"三元四维"人才培养新体系（2019至今）

2019年在继承"三三制"基础上提出，实现从"教改"到"育人"的跨越。

### "三元"——三个育人切入点

| 三元 | 内涵 |
|------|------|
| **德** | 以德导航，铸造以德为先的成长模式，强调家国情怀和社会责任感 |
| **知** | 知识传授与能力培养，夯实专业基础与通识素养 |
| **行** | 实践创新与知行合一，推动科研训练与创新创业 |

### "四维"——四个育人维度

| 维度 | 内容 |
|------|------|
| **学习维度** | 个性化适应性学习模式——多样选择、个性学习、多元出口 |
| **成长维度** | 内涵式层次化成长模式——知识学习→能力提升→以德导航 |
| **育人维度** | 融合式全面型育人模式——育教融合、通专融合、科教融合、本研融合、学科交叉 |
| **环境维度** | 通达式全方位环境模式——构建师生共同发展生态 |

### "新三三制"改革（2025年）

推进AI驱动的因材施教新模式，实施"四个融通"——专业教育与通识教育协同、科技教育与人文教育协同。

### 培养总目标

培养肩负时代使命、具备全球视野、推动科技创新、引领社会发展的**拔尖领军人才**和**优秀创新创业人才**。

**核心素养框架：**
- 强烈的家国情怀与社会责任感
- 广博的知识技能与卓越的专业素养
- 批判性思维与问题解决能力
- 探究精神与创新创造能力
- 合作精神与领导能力
- 有效沟通能力与全球素养

## 四、通识教育体系

### 1. 人工智能通识核心课程——"1+X+Y"三层次

全国首创，由谭铁牛院士、周志华院士领衔。

| 层次 | 名称 | 说明 |
|------|------|------|
| **"1"** | AI通识核心课 | 《人工智能通识与应用》，2025级本科生必修（1学分），谭铁牛书记开讲 |
| **"X"** | AI素养课 | 54门已立项，零基础可学，各学科与AI交叉 |
| **"Y"** | AI前沿拓展课 | 29门已立项，各学科与AI深度融合，进阶实践 |

教学形式：集体授课 + 小班主题研讨 + 实习实践 + AI助教

### 2. 悦读经典计划

全体本科生须获**至少1个"悦读经典计划"学分**，计入"人文与社会科学"模块。

以《南大读本》（上、下册）为导引，近百位"悦读导师"团队，分六大模块：
- 文学与艺术
- 哲学与宗教
- 经济与社会
- 自然与生命
- 全球化与领导力
- 历史与文明

另设DIY研读课（小班研讨），分通识核心类（跨学科）和专业核心类（单学科深入）。

### 3. "科学之光"系列课程

全体本科生须获**至少1个"科学之光"学分**，计入"自然科学与技术"模块。

| 类型 | 师资 | 形式 |
|------|------|------|
| **"大科光"** | 院士或长江学者领衔 | Lab参访 + 讲座 + 研讨（每门120名额，抽签） |
| **"青年科光"** | 留学归国青年学者 | 小班化（≤15人/教师），互动≥16小时/学期 |

示例：谢然《有趣的糖科学史》、董昊《计算显微镜》、《认识宇宙》

## 五、本科生导师制

- 每位本科生**均配备导师**
- 每年约300多名教师加入新生导师队伍
- 覆盖思想价值引领、学业指导与规划全方位引导

## 六、综合评价成绩单

以三维向量建立多元评价机制：
1. 课程类学习成绩
2. 非课程类学习（社会认知、实践动手、沟通领导等能力）
3. 思想品德、学术诚信
